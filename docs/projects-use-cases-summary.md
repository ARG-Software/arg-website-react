# ARG Software Project & Use-Case Summary

## Mojaloop

ARG joined the core Mojaloop vNext team to help rebuild the open-source payment switch from the ground up. The work focused on microservice architecture, zero-trust service boundaries, payment-message compatibility, transaction orchestration, settlement flows, and production-grade QA. The result is a more secure, scalable, and bank-ready foundation capable of supporting high-volume payment processing across banks, wallets, and payment providers, with 2,000+ transactions per second and 99.9%+ uptime.

## People's Clearinghouse

ARG helped extend Mojaloop vNext into a community-owned payments ecosystem for Mexico's social financial sector. The project connected clearinghouse flows, participant systems, mobile transfers, QR merchant payments, admin confirmations, and live demo infrastructure. At Interledger Summit 2025, the platform processed 1,000+ live showcase transfers with zero failures, while underlying APIs demonstrated ~2,000 requests per second and ~770ms average ledger transfer times.

## Dokutar

ARG redesigned Dokutar's backend after the compliance platform outgrew its original Node and WordPress-based architecture. The work included database migration, multitenancy, tenant-aware workflows, stronger security controls, and a backend foundation built for compliance-heavy operations. Dokutar can now support multiple client workspaces, process 50,000+ documents monthly, and scale without compromising tenant privacy.

## Sky Tracks

ARG helped SkyTracks launch a more reliable version of its browser-based music production platform. The project included an Angular frontend migration, implementation of the redesigned interface, latency reduction, external instrument connectivity, feature stabilization, QA, and release support. The platform now supports 50,000+ online tracks, 1,000+ collaborative sessions, and smoother real-time creative workflows for musicians.

## TV Cine

ARG helped TV Cine evolve its premium entertainment platform across the user-facing website, backend API layer, and internal backoffice systems. The work included Vue frontend improvements, backend refactoring beyond the original Strapi limitations, new API functionality, admin workflow improvements, QA, and feature delivery. TV Cine gained a more flexible platform foundation supporting 10,000+ active subscribers, 5,000+ content titles, and 99% platform availability.

## Royalty Flush

ARG built a backend royalty-processing engine for North Music Group to replace spreadsheet-heavy royalty workflows. The platform handles CWR file ingestion, royalty data organization, obligation calculation, reporting, and QA around calculation accuracy. Royalty Flush moved critical royalty operations away from fragile Excel macros into a repeatable backend workflow supporting 30+ CWR file types and 1,000+ music catalogs.

## Vector

ARG rebuilt Vector from a non-functional crypto trading prototype into a working financial product. The work included backend rebuild, secure authentication, frontend-to-backend integration, live exchange connectivity, automated buy/sell execution, strategy management, and QA across trade execution and dashboard state. Vector became a live trading platform with 5+ exchange integrations, under-10ms average execution time, and 24-hour daily monitoring.
